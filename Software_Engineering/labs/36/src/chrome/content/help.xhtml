enconv: Character encoding converter suite

Author

Vladimir Rutsky, 2010

Introduction

Encoding Converter ("enconv") is Mozilla Firefox extension intended for interactive text encoding conversion.

Usage

  1. Select text in Firefox.
  2. Start "Character encoding" context menu item. Encoding Converter window will appear with selected text as input. (Also you can edit input text directly into Encoding Converter window.)
  3. Try selecting different encoding in "Apply conversion"'s lists "From" and "To". Notice that result shown in lower part of window immediately.
  4. At any time you can move acquired result into input textbox by pressing "Use Result As Input". This is usefull for pipeline conversions, when input text must be converted more than one time.
  5. If manual conversion don't give result you can try to automatically detect conversion leading to readable text by pressing "Detect Conversion". It use frequencies table for conversion detection, by default it is for Russian technical language (custom frequency table can be loaded by pressing "Load Frequencies Table" button).

Supported platforms

Currently only Linux x86_64 supported (Firefox OS/ABI string: Linux_x86_64-gcc3).
